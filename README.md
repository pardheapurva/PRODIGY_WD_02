# Modern Animated Stopwatch Web Application

A modern, interactive stopwatch web application built using **HTML, CSS, and JavaScript**.  
The project features a clean aesthetic UI, smooth animations, and essential stopwatch functionalities such as start, pause, reset, and lap tracking.

This project was developed to strengthen frontend development skills and understand real-time JavaScript timer logic.

## ⚙️ How It Works

The stopwatch works using JavaScript's `setInterval()` function.

1. When the **Start button** is clicked, a timer begins running.
2. Every second, the timer updates the seconds counter.
3. Once seconds reach 60, minutes increase.
4. Once minutes reach 60, hours increase.
5. The **Pause button** stops the timer.
6. The **Reset button** clears the timer and lap history.
7. The **Lap button** records the current time.
